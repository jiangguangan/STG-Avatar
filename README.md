## 📝 About STG-Avatar / 项目简介

**STG-Avatar** 是一种基于 3D 高斯溅射（3DGS）的高保真、可驱动人类 Avatar 重建框架。针对现有方法在处理非刚性物体（如衣物变形）和高动态区域（如快速移动的四肢）时的局限性，我们提出了一种创新的解决方案。
